# Claude / Agent Reference

This is a monorepo of personal data visualization mini-projects. Read this before making changes.

## Repo structure

```
/data-viz-projects          ← repo root, git initialized here
  README.md
  CLAUDE.md                 ← this file
  .gitignore
  /health-inequality-dashboard/   ← first project (complete)
  /(future-project-name)/         ← each project is self-contained
```

Each subfolder is fully self-contained — no shared dependencies between projects. Any one can be deployed independently (GitHub Pages, Netlify, etc.).

## Per-project structure (health-inequality-dashboard as template)

```
/project-name
  index.html
  style.css
  script.js
  fetch_and_clean.py    ← one-time data fetch/clean script (Python)
  README.md             ← project-specific docs, data source credit, deploy instructions
  /data
    data.json           ← generated by fetch_and_clean.py, committed
    years_available.json
```

## Stack

- **Data pipeline**: Python (`requests` + `pandas`), run once, outputs static JSON into `/data`
- **Frontend**: Vanilla D3.js loaded from CDN, no build step, no bundler
- **No shared deps**: do not introduce a root-level `package.json` or shared Python env

## Commit discipline

Commit at logical checkpoints within each project:
1. After repo scaffold (root README + .gitignore)
2. After data fetch/clean script works and data files are generated
3. After static first-frame render works
4. After animation and interaction work
5. Final "Complete [project] v1" commit

## Data source conventions

- World Bank Open Data (WDI) — no API key required
- Always credit "Data: World Bank Open Data, World Development Indicators (CC BY 4.0)" in the page footer and project README
- Raw API responses are NOT committed; only the cleaned output JSON is

## D3.js conventions used in this repo

- Load data with `d3.json()` from a relative `./data/data.json` path
- Bubble radius uses `d3.scaleSqrt()` — never linear radius (distorts area perception)
- Health expenditure x-axis uses log scale (`d3.scaleLog()`)
- Year animation uses `d3.interval()` (clearable on pause), transitions at ~750ms/step
- Income group colors: colorblind-safe categorical palette (see `script.js` in health-inequality-dashboard)

## Local development

Browsers block `fetch()` on `file://` URLs. To test locally:

```bash
cd health-inequality-dashboard
python -m http.server 8000
# then open http://localhost:8000
```

Production hosting (GitHub Pages, Netlify) does not have this issue.
